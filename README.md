# ChainCerty

ChainCerty is a decentralized platform for verifying and showcasing skills using blockchain technology. This project leverages Rust (via the ink! framework) for the smart contract, and a responsive HTML5/CSS3 frontend for user interactions.

## Features

- Decentralized skill verification for users.

- Blockchain-based data storage for security and immutability.

- Modern, responsive user interface.

## Requirements

- [Rust](https://www.rust-lang.org/) with `cargo` and `rustup`

- [ink!](https://ink.org/) [CLI](https://cli.org/) for Substrate smart contract development

- A modern web browser
